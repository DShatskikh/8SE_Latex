\begin{frame}{Площадка публикации}
\textbf{ACM/IEEE International Conference on Automated Software Engineering (ASE)}

\vspace{0.5cm}

\begin{itemize}
  \item Одна из ведущих международных конференций в области автоматизации разработки программного обеспечения
  \item Организуется при поддержке \textbf{ACM} и \textbf{IEEE}
  \item Фокус: 
  \begin{itemize}
    \item автоматическая генерация кода
    \item тестирование и верификация ПО
    \item формальные методы
    \item применение искусственного интеллекта в разработке
  \end{itemize}
  \item Высокий уровень отбора статей (строгое рецензирование)
  \item Публикует передовые исследования и практические решения
\end{itemize}

\end{frame}
